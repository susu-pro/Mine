개인정보 처리방침

최종 업데이트: 2026년 1월 22일

요약: Mine은 귀하의 개인 데이터를 수집, 저장 또는 전송하지 않습니다. 모든 데이터는 귀하의 기기에 저장됩니다.

1. 수집하는 정보

없음. Mine은 개인 정보, 사용 데이터 또는 분석 데이터를 수집하지 않습니다.

2. 데이터 저장

Mine으로 열거나 편집하는 모든 파일은 로컬 기기에 저장됩니다. 당사는 귀하의 파일에 접근할 수 없으며, 어떤 서버에도 업로드되지 않습니다.

3. 제3자 서비스

Mine은 제3자 분석, 추적 또는 광고 서비스를 사용하지 않습니다.

4. 네트워크 접근

앱은 인터넷 연결 없이 작동합니다. 네트워크 접근은 문서 내의 외부 링크를 명시적으로 클릭할 때만 사용됩니다.

5. 아동 개인정보 보호

데이터를 수집하지 않으므로 Mine은 모든 연령의 사용자에게 안전합니다.

6. 정책 변경
